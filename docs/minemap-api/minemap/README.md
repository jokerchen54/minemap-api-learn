## 前言
大家可能有疑问，为什么我们要做这个东西呢，我们不是已经有了minemap的官方api文档了吗。对的，不过有些时候我们可能会忽略掉一些使用的api或者配置项，导致在实现同一个效果时，写的代码可能会很冗长，可是用另外的api实现只需要简单的几行。所以为了让大家对minemap引擎有一个一致的认知，特此开展了这文档的编写。

## 设计概要
- 针对每一个api、每一个配置项做补充说明。通常通过demo代码进行说明。
- 标记重点内容、日常使用中需要注意的地方
- 对比mapbox相同的api
- 对于一些常用效果，列出最佳实践供参考
